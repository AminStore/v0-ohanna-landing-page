# 🌅 OHANNA Storefront Client

This directory houses the frontend web interface for the **OHANNA** Egyptian Streetwear platform.

---

## 🛠️ Tech Stack

* **Framework & Build**: React (v18.x) + Vite (v5.x)
* **Type Safety**: TypeScript
* **Styling**: Tailwind CSS + Sass + Shadcn UI
* **State & Fetching**: React Query (TanStack)
* **Client Translation / Localization**: i18next (supports English/Arabic/French)
* **Build Target**: Fully compiled production assets served via high-performance CDNs or Nginx

---

## 📁 Project Directory Layout

```
ohanna/
├── src/
│   ├── api/              # Custom fetch wrappers and endpoints
│   │   └── generated/    # Autogenerated typescript query bindings (Orval)
│   ├── components/       # Storefront components (cart, order tracking, UI cards)
│   ├── contexts/         # Theme controls and translations
│   ├── App.tsx           # Primary routing and initialization layout
│   └── main.tsx          # Client bundle entry point
├── public/               # Static streetwear imagery assets
├── vite.config.ts        # Vite configuration script
└── tailwind.config.js    # Styling layout directives
```
